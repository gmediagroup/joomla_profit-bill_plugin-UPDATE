ВАЖНО:
Плагин работает на PHP5
Разрабатывался для версии joomla 1.6


Принцип работы плагина:
Этот плагин позволяет обеспечить платный доступ к чему-либо на вашем сайте. 
В ответ на присланное смс-сообщение пользователю приходит короткий текстовый пароль (ключ), 
с помощью которого он может получить доступ к тому, что вы закроете этим ключом. 
Вы сами решаете сколько раз или как долго можно использовать пароль для доступа. 

Установка:
1. Установить плагин pb_smsapi.zip через Менеджер расширений (http://nauca.com.ua/poleznoe/kak-ustanovit-lyuboj-plagin-joomla-na-sajt/)
2. Сконфигурируйте плагин: укажите ID SMS API проекта и Секретный код. Остальные параметры опциональны.
3. В проектах SMS API на сайте Profit Bill (http://profit-bill.com/projects/smsapi/) для вашего проекта укажите
* вида ответа - динамический обработчик
* URL обработчика - укажите путь к файлу pb_handler.php на вашем сайте. Путь выглядит как http://<ваш сайт>/plugins/content/pb_smsapi/pb_handler.php


Контент, который размещен между тегами (по-умолчанию pb_smsapi) [pb_smsapi] скрытый текст [/pb_smsapi] будет скрыт.
